# Sorting

There are three different way to sort the code items in CodeNav.

## Sort By Name

Code items are alphabetically sorted and then sorted by file position.

## Sort By Order

Code items are sorted by file position.

## Sort By Type

Code items are sorted by their kind in the order given below and then sorted by name and then sorted by file position.

| #  | Kind |
| -- | ---- |
| 1  | Namespace |
| 2  | BaseClass |
| 3  | Class |
| 4  | LocalFunction |
| 5  | Method |
| 6  | Property |
| 7  | Constructor |
| 8  | Enum |
| 9  | EnumMember |
| 10 | ImplementedInterface |
| 11 | Interface |
| 12 | Variable |
| 13 | Constant |
| 14 | Record |
| 15 | Struct |
| 16 | Event |
| 17 | Switch |
| 18 | SwitchSection |
| 19 | Region |
| 20 | Indexer |
| 21 | Delegate |
